# COMUDE Guadalajara 360

Repositorio con dos productos independientes:

- `comude-dashboard`: plataforma administrativa institucional.
- `comude-mobile`: app ciudadana móvil PWA.

Cada proyecto se instala, ejecuta y despliega por separado.

## Dashboard

```bash
cd comude-dashboard
npm install
npm run dev
npm run build
```

## App móvil

```bash
cd comude-mobile
npm install
npm run dev
npm run build
```
